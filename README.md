# Mrered Resume

个人简历站，基于 Astro 构建。

## 内容重点

- 汇总 Presto-io 项目经验：主应用、官网、文档、模板注册中心、官方模板、脚手架和 AI Skills。
- 展示电气工程及其自动化专业能力、职业教育实践、AI 原生开发、平台工程、DevOps、文档工程和教育治理能力。
- 保留单页静态输出，适合部署到 GitHub Pages、Vercel、Netlify 或任意静态托管服务。

## 开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
pnpm preview
```

构建产物输出到 `dist/`。
